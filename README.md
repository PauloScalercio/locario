# Locario

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.0.7.

## Como rodar?

Abrir a pasta do projeto no terminal e rodar:

1 - npm install

2 - npm start

OBS: É necessário ter node e npm instalados.
